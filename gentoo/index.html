Gentoo

installation instructions

 

 

Creates our custom ebuild directory:

sudo mkdir -p /usr/local/portage/app-misc/peerio-client

 

Make sure your /etc/make.conf refers to that directory

grep /usr/local/portage /etc/make.conf

 

If the previous command don't show some PORTDIR_OVERLAY definition, use the following:

echo 'PORTDIR_OVERLAY="/usr/local/portage"' | sudo tee -a /etc/make.conf

 

Install our custom ebuild, download our Manifest

cd /usr/local/portage/app-misc/peerio-client

sudo wget https://linux.peerio.com/gentoo/peerio-client-1.2.0.ebuild

sudo wget https://linux.peerio.com/gentoo/Manifests

 

You should then be able to unstall our client using emerge

sudo emerge peerio-client

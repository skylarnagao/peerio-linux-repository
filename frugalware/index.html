installation instructions

 

 

If you haven't installed un-official packages yet, you may need to prepare your environment to do so

sudo pacman-g2 -S curl pacman-tools git rsync fakeroot

sudo repoman upd

 

Prepare your build directory

mkdir /var/fst/stable/sources/xapp/peerio-client

cd /var/fst/stable/sources/xapp/peerio-client

wget https://linux.peerio.com/frugalware/FrugalBuild

 

Build your package

makepkg

 

Upon completion, the build process would output a package in your working directory, install it

pacman-g2 -U peerio-client-1.2.0-1-x86_64.fpm

General installation instructions

 

 

Building from sources, there's a few dependencies you will need, such as nodejs and curl or npm

You may refer either to our docker files or our debian package dependencies for an exhaustive list

 

Fetch our code from github, or the latest release archived here, and extract its content

wget https://linux.peerio.com/sources/peerio-client-1.2.0.tar.gz
tar -xzf peerio-client-1.2.0.tar.gz
cd peerio-client/

 

Start building our code using the provided makefile

make

 

If the build process fails, you're probably missing a dependency

Otherwise, you should be able to install our client, running the following:

make install
